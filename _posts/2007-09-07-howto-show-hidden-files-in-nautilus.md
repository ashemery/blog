كثير من الأحيان نحتاج الى غرض الملفات المخفية، ربما نحن أخفيناها وربما نريد الوصول الى ملف معيين من هذه الملفات ولهذا بإمكاننا ذلك في الـ Nautilus من خلال التالي:

الطريقة الأولى أضغط على   ‘Ctrl + H’

الطريقة الثانية هي:

```
Edit Menu -> Preferences
```

بعد ذلك أذهب الى:

```
Views Tab -> Default View -> Show hidden and backup files
```
