السلام عليكم ورحمة الله وبركاته

أخيراً وبعد فترة من العمل، أود الأعلان بشكل رسمي عن [شركة عرب نيكس](http://arabnix.com) لحلول جنو/لينوكس الإحترافية.[شركة عرب نيكس](http://arabnix.com) جائت كناتج لسنوات الخبرة والعمل في هذا المجال وبمختلف القطاعات المهنية. ستقوم الشركة بتقسيم مجال عملها الى ثلاث خدمات وحلول رئيسية:

**حلول [Arabnix](http://arabnix.com) للخوادم** – وهذه تشمل الحلول التالية

```
Domain Name Server (BIND)
Email Server (SMTP/POP3/IMAP)
Proxy Caching Server
Webserver and Hosting Server
FAX Server
Firewall Server
Network File and Backup Server
Network Monitoring Server
Network Time Server
Print Server
Virtual Private Network Server
News Servers (NNTP)
DansGuardian - web content filtering system
Dynamic Host Configuration Protocol (DHCP)
OpenLDAP - Lightweight Directory Access Protocol
```

بالإضافة الى حلول أخرى مخصصة لتلبي إحتياجات عملك.

**حلول [Arabnix](http://arabnix.com) الأمنية** – وهذه تشمل الحلول التالية

```
Basic Server Security
Data and Filesystem Encryption
System Hardening: Linux Services
System Integrity: Config Files
Kernel Hardening and Patching
Firewall Hardening and Auditing.
Intrusion Detection System (IDS) & Intrusion Prevention System (IPS)
```

وأخيراً **حلول [Arabnix](http://arabnix.com) التدريبية** – هذه الحلول هدفها قيادة المتدرب الى أهم الأمور التي يحتاجها للعمل على نظام جنو/لينوكس ولكي تحقق أحتياجاته وإحتياجات الشركة التي يعمل بها. البرنامج المستخدم يتم الإتفاق عليه من قبل الطرفين وذلك لكي نظمن تحقيق أهدافكم من هذا التدريب.

للمزيد من المعلومات والإستفسارات يمكنكم زيارة [الموقع الرسمي](http://arabnix.com) والتعرف أكثر حول تفاصيل كل خدمة/حلول، وأيضاً يمكنكم مراسلتنا على البريد التالي:

**للمبيعات:** sales@arabnix.com

**للإستفسارات:** support@arabnix.com
