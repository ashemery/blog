لتحسين عمر المحمول Laptop نقوم بتفعيل خاصية تسمى Laptop mode … هذه الخاصية متوفرة في الكيرنل 2.6.x وتساعد على التحكم بالــ IO Subsystem … يعني يصبح لينوكس يقلل من عدد طلبات الــ IO … ليصبح يعمل عند الحاجة …  وبهذا يصبح الإستهلاك للبطارية لديك أقل وتزداد عمراً :)

الآن لتعرف هل هذه الخاصية مفعلة على المحمول الذي لديك أم ﻻ … نفذ التالي:

```
cat /proc/sys/vm/laptop_mode
```

إذا كان الناتج = 0 ؟ هذا يعني إنه غير مفعل … ولتفعيله نفذ التالي:

```
echo 5 > /proc/sys/vm/laptop_mode
```

الآن لتثبيته لكي يعمل حتى بعد عملية إعادة التشغيل restart للجهاز نفذ التالي:

```
vi /etc/sysctl.conf
```

ومن ثم ضع داخله السطر التالي:

```
vm.laptop_mode = 5
```

احفظ الملف ومن ثم نفذ التالي للتأكد فقط:

```
sysctl -p
```

الآن مبروك تفعلت عندك الخاصية هذه …
