السلام عليكم ورحمة الله وبركاته …

بعض الأحيان نريد أن نقوم بتشغيل برنامج معيين بصلاحيات root … ولهذا كنا ندخل على الــ Terminal ومن ثم نطبق التالي:

```
su -
```

ومن ثم نشغل البرنامج المطلوب وليكن nautilus كما في التالي:

```
nautilus &
```

أو طريقة أخرى هي إستعمال الــ sudo للبرامج مثلا:

```
sudo apt-get install PROGRAM
```

طيب وهل يوجد طريقة أخرى ؟ نعم يوجد وذلك من خلال الأمر gksudo أو gksu … عندما تريد تشغيل أي برنامج ذو واجهة رسومية كل ما عليك فعله هو التالي:

Alt + F2 ومن ثم أكتب:

```
gksudo PROGRAM
```

أو

```
gksu PROGRAM
```

أي نكتب gksudo أو gksu وإسم البرنامج … وهكذا سيعمل بواسطة صلاحيات root …

طبعا للبرنامج هذا ميزات أخرى بإنك تستطيع أن تحدد يعمل بواسطة أي مستخدم … وذلك من خلال التالي:

```
gksu -u binary PROGRAM
```

أو

```
gksudo -u binary PROGRAM
```

حيث سيقوم بتشغيل البرنامج  PROGRAM بصلاحيات المستخدم binary … لمعرفة المزيد ؟
