السلام عليكم ورحمة الله وبركاته …

أول حاجة تعلمتها من خلال عملي على Ubuntu Ultimate هي كيفية تنصيب البرامج من نوع .deb وأليكم كيفية التعامل مع هذا النوع بإختصار، وأيضاً أنا سأضع الطرق بدون الأمر Sudo ﻷنني أقوم بهذه العمليات من خلال المستخدم root ولكن إن كنت تريد أن تقوم بها بنفس المستخدم العادي ﻻ تنسى أن تضع Sudo في بداية كل أمر.

لتنصيب ملف قم بعمل التالي:

```
dpkg -i pkg_file.deb
```

لحذف البرنامج الذي بالأعلى:

```
dpkg -r pkg_file
```

لمعرفة معلومات عن البرنامج:

```
dpkg -l pkg_file
```

لتحويل برنامج من نوع .rpm الى .deb :

```
alien pkg_file.rpm
```

في المثال أعلاه ﻻ حاجة لوضع الخيار -d وذلك ﻷنه هو الـ Default ولكن لو كنت تريد تحويل العكس قم بوضع -r لتحويله الى .rpm … وعليك بالــ man alien لمزيد من المعلومات …
