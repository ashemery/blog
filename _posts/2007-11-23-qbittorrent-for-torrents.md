![](/assets/img/posts/2007/qTorrentbanner.png)

أليكم برنامج تورنت جديد إسمه qBittorrent تم برمجته على QT4 …

لتنصيبه بالنسبة لمستخدمي  Ubuntu Gusty نفذ التالي:

```
vi /etc/apt/sources.list
```

وبعدها ضع بداخله السطرين هذول:

```
deb http://hydr0g3n.free.fr/qbittorrent/gutsy/ ./
```

```
deb-src http://hydr0g3n.free.fr/qbittorrent/gutsy/ ./
```

قم بحفظ التغييرات ومن وبعدها أخواني نفذوا أمر تحديث المكتبات:

```
apt-get update
```

وبعدها نفذ أمر التنصيب:

```
apt-get install qbittorrent
```

ثم لتشغيله:

```
Applications—> Internet—> qBittorrent
```
