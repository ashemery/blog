السلام عليكم ورحمة الله وبركاته ..

يقوم هذا البرنامج بالبحث عن البرامج التي تعمل بشكل غير إعتيادي أثناء كون جهازك في وضعية عدم الإستعمال أو السبات أي Idle … من ميزات البرنامج هو:

**أوﻻ:** يعرفك بمدى إستعمال نظامك التشغيلي لمميزات توفير الطاقة.

**ثانيا:** يساعد المطورين على تجربة تطبيقاتهم لكي يصلوا الى أفضل قدر ممكن من إستهلاك الطاقة.

**ثالثا:** يعطيك إقتراحات لكيفية عمل Tuning لتصل الى أقل إستهلاكاً للطاقة.

**رابعا:** يساعدك على إكتشاف ما هي البرامج التي تقوم بعمل الكثير من الـ Interrupts وتجعل البرنامج يتحول من حالة السبات الى حالة العمل بإستمرار.

المطورين لهذا البرنامج أكتشفوا بإنه أكثر الموديوﻻت التي تسبب إستهلاك في الطاقة هي:

```
snd_intel8x0m
fglrx
radeon
ipw*
bcm*
hdaps
ibm_acpi
i8042
nvidia
```

**لتحميل البرنامج:**

توزيعة Ubuntu وأشقاءها:

```
apt-get update;
apt-get install powertop;
```

توزيعة Fedora:

```
yum install powertop
```

توزيعات أخرى —–> [أضغط هنا](http://www.lesswatts.org/projects/powertop/download/powertop-1.9.tar.gz)

ومن ثم

```
tar -zxf powertop-1.9.tar.gz;
cd powertop-1.9;
make;
make install;
```

لتشغيل البرنامج نفذ التالي:

```
powertop
```
