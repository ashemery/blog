السلام عليكم ورحمة الله وبركاته

وأنا أتصفح مدونة فريق Google الخاص بمجال الأمنية وجدت [Keyczar](http://www.keyczar.org/) والذي هو عبارة عن صندوق عدة toolkit للتشفير، مفتوح المصدر يفيد المطورين للبرامج في إستخدام ميزة التشفير فيها … [Keyczar](http://www.keyczar.org/) يدعم التشفير المتماثل/التماثلي symmetric والغير متماثل asymmetric … ومن ميزاته:

*  A simple API

* Key rotation and versioning

* Safe default algorithms, modes, and key lengths

* Automated generation of initialization vectors and ciphertext signatures

* Java and Python implementations (C++ coming soon)

* International support in Java (Python coming soon)

تم تطوير [Keyczar](http://www.keyczar.org/) بواسطة بعض أعضاء فريق Google للأمنية ([Google Security Team](http://googleonlinesecurity.blogspot.com/))، ورخصته هي رخصة [الأباتشي2](http://www.apache.org/licenses/LICENSE-2.0.html) …

إن شاء الله يفيدكم يالمبرمجين … حبيت نشر الأمر لعل وعسى يفيد أحدكم …
