السلام عليكم ورحمة الله وبركاته

لعرض القواعد الموجودة لديك قم بالدخول الى خادم MySQL ومن ثم نفذ الأمر التالي:

```
show databases;
```

سيتم عرض القواعد الموجودة عندك … من ظمن هذه القواعد ستجد قاعدة إسمها mysql وهي خاصة بالصلاحيات والأمور المتعلقة بالمستخدمين الخاصين بالـ MySQL أي هي للـ User Access Data …

الآن لكي تستطيع أن تعمل على قاعدة معينة عليك إختيارها ويتم ذلك من خلال الأمر:

```
use dbname;
```

الآن لرؤية الجداول الموجودة في هذه القاعدة نفذ الأمر:

```
show tables;
```

لعرض تفصيلات أو خيارات جدول معيين نفذ الأمر:

```
describe tablename;
```
