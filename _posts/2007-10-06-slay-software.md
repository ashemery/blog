السلام عليكم ورحمة الله وبركاته …

برنامج Slay مفيد في عمل Kill لجميع البرامج التي تعمل بإستخدام مستخدم معيين … كل ما عليك فعله هو تنفيذه كالتالي:

```
slay -signal-number user-name
```

لتنزيله في Ubuntu:

```
apt-get install slay
```
