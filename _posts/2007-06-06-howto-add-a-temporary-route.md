السلام عليكم ورحمة الله وبركاته …

بعض المرات لأغراض الإختبار لأمر معيين في الشبكة تحتاج الى وضع مسار مؤقت من خلال سطر الأوامر. يمكن عمل ذلك من خلال الأمر route لنأخذ مثال:

```
route add -net 10.0.0.0 netmask 255.0.0.0 gw 192.16.0.1 dev eth0
```

حيث قمنا بإضافة مسار لأي بيانات يريد الذهاب الى الشبكة 10.0.0.0 أن يذهب الى المنفذ Gateway الذي رقمه 192.16.0.1 ومن خلال البوابة eth0 …

ولإضافة بوابة رئيسية بإمكانك عمل ذلك من خلال الأمر التالي:

```
route add default gw 192.168.0.1 eth0
```

أيضاً تذكرت الحين شيء آخر … لو تريد تغيير الـ IP الخاص بجهازك مؤقتاً بإمكانك ذلك من خلال:

```
ifconfig eth0 192.168.0.11
```

نراكم في درس بسيط آخر قريباً إن شاء الله
