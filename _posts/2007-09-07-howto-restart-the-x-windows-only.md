السلام عليكم ورحمة الله وبركاته …

من الأمور الجميلة والقوية في لينوكس هو فصل نظام الواجهة الرسومية عن النظام الأساسي للينوكس Core System وتعتبر الواجهة الرسومية نظام منفصل بحد ذاته إسمه X أو X11 أو XFree86 …  لعمل إعادة تشغيل للواجهة الرسومية في حالة حصل تهنيق للنظام بسببها أعمل أحد الطرق التالية:

أضغط على كل من ‘Ctrl + Alt + Backspace’

أو في أوبنتو

```
sudo /etc/init.d/gdm restart
```

أو في سوزي

```
rcxdm restart
```

أو من خلال

```
/etc/init.d/xdm restart
```

أو من خلال عمل Kill لها من الـ Virtual Terminal ومن ثم تشفيلها مرة أخرى … أو الذهاب الى Run Level 3 ومن ثم العودة الى Run Level 5 … يعني بالنهاية حلول وطرق كثيرة …

مع لينوكس مش حتقدر تغمض عينيك  ::18
