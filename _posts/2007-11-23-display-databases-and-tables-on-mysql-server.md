السلام عليكم ورحمة الله وبركاته …

إذا كنت تريد ان تعرض قواعد البيانات الموجودة على سيرفر MySQL من خلال سطر الأوامر ما عليك إﻻ تتبع ما هذه التدوينة البسيطة … أول شيء من الشل جرب نفذ التالي:

```
mysql
```

سيقول لك:

```
ERROR 1045 (28000): Access denied for user 'binary'@'localhost' (using password: NO)
```

وذلك ﻷنه يجب أن يقوم المستخدم binary بالإتصال بالسيرفر من خلال إسم مستخدم وكلمة سرية له … أذن علينا تغيير الأمر لكي يكون كما يلي:

```
mysql --user=binary --password=MYPASSWORD
```

طبعاً ضع أنت بدل binary إسم المستخدم الذي تريد أن يتصل بالسيرفر MySQL وضع بدل  MYPASSWORD كلمة السر الخاصة بك … طبعاً ﻻ تنسى بإنه يجب أن يكون المستخدم هذا أصلا موجود على السيرفر ومصرح له الدخول وما الى ذلك … وهذه القضية إفترضت إنها موجودة … الحين بعد تنفيذ الأمر بالأعلى سيظهر لك التالي:

```
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 87
Server version: 5.0.45 Source distribution

Type 'help;' or '\h' for help. Type '\c' to clear the buffer.

mysql>
```

الآن لعرض قواعد البيانات الموجودة على السيرفر نفذ التالي:

```
show databases;
```

ستظهر لك جدول فيه جميع القواعد التي لديك … الآن نريد أن نرى الجداول في إحدى هذه القواعد نفذ الأمر التالي:

```
use test;
```

هذا لكي يتم إختيار القاعدة التي إسمها test والتي تأتي بالأساس بعد تنصيب سيرفر الـ MySQL … بعدها نفذ التالي:

```
show tables;
```

سوف تعرض لك جميع الجداول الموجودة في القاعدة test … هكذا قمنا بعرض القواعد والجداول التي فيها … للخروج أكتب exit فقط …
