السلام عليكم ورحمة الله وبركاته …

منذ فترة لم أكتب عن قواعد البيانات على كلاً … هذا درس خفيف وسريع عن كيفية إسترجاع قاعدة البيانات التي لنفترض قمت بأخذها قبل فترة والحالية رأيت فيها مشكلة … الحل بسيط جداً … أولاً تأكد من تشغيل الـ MySQL من خلال التالي:

```
/etc/init.d/mysqld start
```

بعد ذلك قم بالدخول على النظام حق الـ MySQL من خلال التالي:

```
mysql -u root -p
```

أدخل كلمة السر حق المستخدم root الخاص بالـ MySQL وستكون وصلت الى هنا:

```
mysql>
```

بعد ذلك لنفرض تريد إستعادة قاعدة بيانات إسمها my_blog عليك أوﻻً أن تعمل قاعدة بالإسم نفسه من خلال:

```
mysql> CREATE DATABASE my_blog;
```

بعد ذلك أخرج من الشل الخاص بالـ MySQL من خلال الأمر:

```
mysql> quit;
```
