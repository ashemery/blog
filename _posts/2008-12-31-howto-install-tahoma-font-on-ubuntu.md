**السلام عليكم ورحمة الله وبركاته**

بعد أن قام [عبدو](http://br4v3-h34r7.com) بعمل التعريب والتضبيط للثيم الحالي للزون … وجدت خطوطه عندي غير جميلة فقلت له يارجل قم بتغيير الخطوط فهي غير جميلة عندي … قال لا هي رائعة عنده … وبعد ان تبادلنا بالصور … تبيين إنه يستعمل الخط Tahoma وهذا ما لا استعمله أنا … ولهذا أرسله لي وقمت بتركيبه أيضاً وذلك لرغبته في أن أقوم بذلك … ولأن الرجل مو وجه فشل ما فشلته وقمت بتركيبه على الجهاز … ولهذا بس حبيت أكتب الخطوات السهلة التي عملتهم لتركيبه على الجهاز وهي كالتالي:

**أولاً**: قم بتحميل الخطوط TrueType من خلال تنفيذ الأمر:

```
sudo apt-get install msttcorefonts
```

**ثانياً**: قم بتحميل الخط Tahoma من [هنا](http://binary-zone.com/files/tahoma.ttf) وذلك لأنه الحزمة msttcorefonts تفتقر الى وجودها ولكنها تفيدك بوجود خطوط أخرى جميلة …

**ثالثاً**: أفتح الطرفية وقم بنسخ الخطوط من محل ما هي فيه الى المجلد الخاص بالخطوط أو الى المجلد الخاص بالمستخدم الخاص بك إذا لم ترغب بوضع الخطوط لتستعمل على النظام بالكامل … يعني:

```
sudo cp tahoma.ttf /usr/share/fonts/truetype/msttcorefonts/
```

أو إذا كما قلنا تريدها لك فقط:

```
sudo cp tahoma.ttf ~/.fonts/
```

**رابعاً**: إما تقوم بتسجيل الخروج Logout ومن ثم تقوم بعمل Login على النظام … أو تقوم بعمل إعادة توليد للكاش الخاص بالخطوط من خلال الأمر:

```
sudo fc-cache -fv
```

وهكذا خلاص أصبحت الخطوط Tahoma راكبة مع خطوط أخرى مثل Arial و Comic_Sans_MS و Courier_New وغيرهم الكثير …

ودمتم بود …
