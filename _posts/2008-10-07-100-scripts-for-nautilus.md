السلام عليكم ورحمة الله وبركاته

![](/assets/img/posts/2008/nautilusscripts1.png)

جايبلكم اليوم بضاعة ولا أروع ولا اجمل … أكثر من 100 سكربت لتسهيل عملك على Nautilus … كل ما عليك هو إضافتهم الى الـ Nautilus وسيساعدوك في الكثير من المهام … السكربتات مقسمة الى الأقسام التالية:

```
Archiving
Executing
File info
File processing
Multimedia
File System Management
System Configuration
```

كل الي عليك تسويه هو:

```
cd ~/.gnome2/nautilus-scripts
wget -c http://g-scripts.sourceforge.net/nautilus-scripts.tar.gz
tar zxvf nautilus-scripts.tar.gz
```

وبعد ذلك ستجدهم على الضغطة اليمين للفأرة كما في الصورة:

![](/assets/img/posts/2008/nautilusscripts.png)

طبعاً هذه الخلفية التي على سطح مكتبي هي لطيب الذكر أخوي باسم، وموجودة لمن يريدها من هنا –> [أضغط](https://www.linuxac.org/forum/showthread.php?t=15575)

أنصحكم بتجربتهم رائعين صراحة … ولا تنسوا عددهم تجاوز الـ 100 … يعني محتاج شغل أسبوعين ثلاثة بس تجربهم كلهم …

أيضاً تستطيع إضافة سكربتاتك الخاصة بكل سهولة:

```
mkdir ~/.gnome2/nautilus-scripts/binary
```

وقم بوضع السكربتات الخاصة بك في المجلد الذي عملته

```
cp myscript.sh ~/.gnome2/nautilus-scripts/binary
```

وهكذا ستجد السكربتات هذه في ضغطة زر يمين …

والقادم أجمل إن شاء الله
