السلام علبكم ورحمة الله وبركاته …

معرفة معلومات عامة عن إصدارة النواة المستعملة:

الكثير من الأخوة يتسائلون:

1- كيف أعرف ما هو رقم إصدارة النواة التي أقوم بتشغيلها؟

2- كيف أعرف نوع العتاد الذي أستعمله؟

3- كيف أعرف نوع المعالج الذي أستعمله؟

الجواب:

1- لمعرفة رقم الإصدار حق النواة نفذ الأمر كالتالي:

```
uname -r
```

```
2.6.16.54-0.2.3-smp
```

بإمكانك معرفة رقم الإصدارة للنواة من خلال الأمر التالي أيضاً:

```
cat /proc/version
```

2- لمعرفة نوع العتاد أستعمل الأمر:

```
uname -i
```

3- لمعرفة نوع المعالج نفذ الأمر التالي:

```
uname -p
```

أو

```
cat /proc/cpuinfo
```

موضوع بسيط جداً لكن مفيد للمبتدئين إن شاء الله …
