السلام عليكم ورحمة الله وبركاته …

بعض الآحيان تكون تتصفح المجلدات والملفات التي لديك، وفجأة تريد تنفيذ بعض الأوامر على ذلك المجلد … بدل من الذهاب وفتح الـ Terminal بإمكانك فتح الـ Terminal مباشرة من الـ Nautilus بإستخدام الإضافة nautilus-open-terminal …

حيث عند الذهاب الى File ومن ثم Open Terminal سيقوم بفتح المجلد الذي تتصفحه أنت … وبهذا يسهل عليك العمل … على كل حال لإضافة هذه الإضافة الجميلة كل ما عليك فعله هو:

**SUSE:**

```
smart install nautilus-open-terminal
```

**Ubuntu:**

```
apt-get install nautilus-open-terminal
```
